# Ansible-playbook-Wireguard-xt_wgobfs
Плейбук устанавливает и настраивает Wireguard и модуль ядра Linux xt_wgobfs, который служит для обфускации Wireguard-трафика на сервере и клиенте на ОС Debian. 

**Плейбук автоматически выполняет:** 

1. Генерирование: пары ключей для сервера и клиента, порта сервера и ключа обфускации. 
2. Установку и настройку Wireguard, применение сгенерированных параметров на WG-сервере и WG-клиенте.
3. Установку и настройку модуля ядра xt_wgobfs на WG-сервере и WG-клиенте.
4. Установку iptables, добавление правил для обвускации трафика. 
5. Ping WG-интерфейса клиента с сервера, что бы убедиться, что туннель работает.
